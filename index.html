What is now the new player (karaoke2.html) does not have support for the pusht and popt commands, but does have support for push t and pop t and a couple other push and pop things that you won't actually use and none of the other push and pop things that you would use. New instructions are notext and text, which disable the karaoke words. vol 0 now skips writing to the audio buffer, but still skips the time. It is effectively equivalent to the no pitch argument in this way, although with no, it is not possible to put karaoke words on the screen. Using both notext and vol 0 is equivalent to the no pitch argument.
